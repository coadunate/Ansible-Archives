HDF5 Ansible Role
=========

Ansible role for installing the latest version of HDF5 

Dependencies
------------

 - [coadunate.devtools](https://github.com/coadunate/ansible-role-devtools.git)

Example Playbook
----------------

    - hosts: servers
      roles:
         - { role: coadunate.hdf5 }
